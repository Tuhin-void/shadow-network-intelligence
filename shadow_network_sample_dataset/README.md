# Shadow Network Sample Dataset

This folder contains a small sample preview of the synthetic AML/FinCrime dataset generated by the Shadow Network Intelligence data engine.

Contents:
- entities/: core graph entities
- edges/: graph relationships
- documents/: semantic RAG corpus
- benchmarks/: evaluation + fraud ring metadata
- streaming/: live event simulation payloads
- tigergraph/: graph loader examples

Purpose:
- dashboard development
- GraphRAG testing
- Vector RAG testing
- benchmark prototyping
- visualization testing
