# Diagrams

This directory is reserved for architectural diagrams referenced from
the documents in `10_research/`. ASCII diagrams live inline in those
documents; this folder is for rendered images (PNG/SVG) when added.

The primary architecture diagram is in
[`../../docs/ARCHITECTURE.md`](../../docs/ARCHITECTURE.md) as ASCII.
The investigation event flow diagram is in
[`../06_operational_investigations.md`](../06_operational_investigations.md).

Adding images here is optional — the documents are designed to be
readable without them.
